# Story Mapping

## Novos Tipos de Quiz (yellow)

### Criar Quiz Qualitativo
- Status: [ ]
- Versão: v1.0
- Link: quiz-eval

Com as permissões de criação de conteúdo, quero criar um quiz de pesquisa coletar INFORMAÇÕES QUALITATIVAS dos usuários.

* Perguntas abertas (Open-Ended)
* Sem limite de criação
* Configurar nome da lição e pergunta
* Suporte a formatação rich text

### Responder Quiz Qualitativo
- Status: [x]
- Versão: v1.0
- Link: quiz-eval

Como um usuário matriculado, quero responder as pesquisas qualitativas de opinião.

* Link de redirecionamento
* Até 2000 caracteres
* Retorno ao WhatsApp após envio
* Bloqueio de conclusão sem resposta

### Criar Quiz Quantitativo
- Status: [ ]
- Versão: v1.1
- Link: quiz-eval

Com as permissões de criação de conteúdo, quero criar um quiz de pesquisa coletar INFORMAÇÕES QUANTITATIVAS.

* Múltipla escolha (Single-Select)
* Até 5 alternativas
* Sem alternativa correta (apenas opinião)
* Suporte a formatação rich text

### Responder Quiz Quantitativo
- Status: [x]
- Versão: v1.1
- Link: quiz-eval

Como um usuário matriculado, quero responder as pesquisas quantitativas de opinião.

* Seleção única
* Sem feedback de erro/acerto
* Retorno ao fluxo WhatsApp
* Bloqueio de conclusão sem resposta

### Criar Quiz NPS
- Status: [ ]
- Versão: v1.2
- Link: nps

Quero criar um quiz de pesquisa para coletar o NPS do curso.

* Apenas um por curso
* Escala 0-10 ou 0-5
* Cálculo automático (Promotores - Detratores)
* Classificação automática

### Responder NPS
- Status: [x]
- Versão: v1.2
- Link: nps

Como um usuário matriculado, quero responder ao NPS do curso.

* Seleção de nota na escala
* Sem feedback de certo/errado
* Fluxo obrigatório para conclusão
